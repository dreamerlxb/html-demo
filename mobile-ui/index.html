关于SUI Mobile

SUI Mobile 是阿里巴巴共享业务事业部SDC团队（UED）提供的一套手机UI库。

我们的目的是为手机千牛、百川以及任何手机端的H5页面提供标准的设计规范和前端实现。

SUI Mobile并没有重新发明轮子，我们主要参考了 ratchet 和 Framework7

这里只是一个官方示例，详细文档请参考：http://m.sui.taobao.org/
